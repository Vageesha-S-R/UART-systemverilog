# UART-systemverilog

This project implements a configurable UART in SystemVerilog.

Features:
- TX / RX modules
- Baud rate generator
- TX and RX FIFOs
- Configurable baud rate
- SystemVerilog testbench

Project structure:

rtl/  -> RTL modules
tb/   -> testbench
sim/  -> simulation scripts
docs/ -> architecture diagrams
